## How To Use 

From your command line, clone and run portfolio:

```bash
# Clone this repository
$ git clone https://github.com/irinazay/portfolio

# Go into the repository
$ cd portfolio

# Install dependencies
$ npm install

#Start's development server
$ npm start
```
